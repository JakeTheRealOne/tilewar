# Server documentation - Return codes

After making sure that the server is running, the api.php file will handle fetch GET and return codes according to the nature of the response.

## List of return codes

| Code   | Description             |
|--------|-------------------------|
| 322500 | Success                 |
| 322501 | Unknown action          |
| 322502 | User already registered |
| 322503 | Unknown user            |
| 322504 | Invalid password        |
| 322505 | Internal server error   |
| 322506 | Not authentified        |
| 322507 | Bad input               |
| 322508 | Unkown category         |
| 322509 | Unkown tile             |
